7. 코드 관련 태그 (기술 문서에 필수)

console.log() 함수는 JavaScript에서 디버깅에 사용됩니다.

function greet(name) {
    return `Hello, ${name}!`;
}

const message = greet('World');
console.log(message);

8. 인터랙티브 요소

자주 묻는 질문 1: HTML이란 무엇인가요?

HTML(HyperText Markup Language)은 웹 페이지의 구조를 정의하는 마크업 언어입니다. 텍스트, 이미지, 링크 등의 요소들을 구조화하여 브라우저가 이해할 수 있도록 합니다.

자주 묻는 질문 2: 시맨틱 태그를 왜 사용하나요?

시맨틱 태그는 HTML 문서의 의미를 명확하게 표현하여 검색엔진 최적화(SEO)와 웹 접근성을 향상시킵니다. 또한 코드의 가독성과 유지보수성도 높아집니다.
